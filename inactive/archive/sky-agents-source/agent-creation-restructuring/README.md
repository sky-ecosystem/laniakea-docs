# Agent Creation & Restructuring

Mechanics for how Synomic Agents are brought into existence and reorganized. All creation and restructuring actions for rank 2–3 agents require a Guardian Accord — a guardian agent must support and trigger the action on the creator's behalf. Rank 1 agents (Core Controlled, Recovery) are created directly by Core Council governance.
