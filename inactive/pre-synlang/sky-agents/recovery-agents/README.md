# Recovery Agents

Rank 1 crisis agents administered by the Core Council. Recovery Agents are activated when a Guardian collapses or is implicated in misconduct — they take over the affected agent tree and manage liquidation, restructuring, or wind-down.

Recovery Agents are temporary and tokenless. They dissolve after the crisis is resolved.
