# Core Controlled Agents

Rank 1 agents directly administered by the Core Council. Core Controlled Agents are general-purpose operational vehicles — in the short term they manage legacy protocol positions (Morpho vaults, Aave pools, SparkLend, etc.) that were previously described as "Core Halos." In the long term they serve any operational need the Core Council identifies.

Core Controlled Agents are tokenless and have no growth asset status.
